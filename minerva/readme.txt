============= INSTALLATION ================

If you are on OSX or Ubuntu:

 Source the corresponding environment:
  
  source osx.sh
  
   OR
  
  source ubuntu-64-environment.sh

Run:
 
 scons install=true

If you are on another operating system, follow the
examples in osx.sh or ubuntu-64-environment.sh to
set up the environment for your system.

============= RUNNING =====================

Follow the built-in help messages.

 minerva-classifier -h
